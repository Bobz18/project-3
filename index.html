How Do Climate Scenarios Change Global Temperature by 2100?

This interactive D3 visualization compares projected global temperature warming under different CMIP6 climate scenarios.

Write-up

Design Rationale

We chose a line chart because the main goal is to compare how global temperature changes over time under different climate scenarios. The x-axis represents year, and the y-axis represents warming since 2015. Color separates the scenarios so viewers can compare low-, medium-, and high-emission futures.

Data Transformation

The CMIP6 data was processed in Python before being loaded into D3. We calculated global mean temperature, converted the data into annual averages, and measured warming relative to 2015. The cleaned result was exported as a CSV file.

Interaction

The checkboxes let users filter scenarios. The year slider lets users focus on shorter or longer time ranges. Tooltips provide exact values when users hover over the points.

Development Process

We first explored the climate data in Python, then selected scenario and year as the main variables for comparison. After cleaning the data, we exported it to CSV and built the final interactive visualization using D3.js.
